Create a new component with the project's conventions:

1. Ask for: component name, directory (src/components/ subdirectory), whether it needs client-side interactivity
2. Create the component file with proper imports and TypeScript types
3. If client-side, add "use client" directive
4. Use Shadcn components from @/components/ui and Tailwind classes
5. Add basic prop types interface
6. Export as named export
7. Use @/* path aliases for all imports
